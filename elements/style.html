<style>

English Base: Style
Spanish translation/Traducción en Español: Estilo

Definición

La etiqueta <style> se usa para asignar reglas de formato, como fuente, color y espaciado, a los elementos dentro de un documento. Esta etiqueta se puede usar en un solo elemento o globalmente. Por defecto, cualquier etiqueta <style> usada en un documento HTML debe tener instrucciones de estilo específicas que se encuentran en el archivo de CSS.

Definition

The <style> tag is used to assign formatting rules, such as font, color and spacing, to elements within a document. This tag can be used on a single item or globally. By default, any <style> tag used in an HTML document should have specific style instructions found in the CSS file.

¿Cómo recordar?

Así como alguien es capaz de escoger su estilo de ropa, la etiqueta <style> (estilo) le permite al autor escoger la aparencia de los elementos en un documento.

How to Remember

Just as someone is able to choose their style of clothing, a <style> tag lets the author choose how items in a document look.

    
      <head>
        <h1><style>This content is styled in CSS<style><h1>
      </head>
