<html>

English Base: Hyper Text Markup Language
Spanish translation/Traducción en Español: Lenguaje de Marcas de Hiptertexto

Definición

El elemento <html> es el elemento raíz de un documento HTML. Generalmente, es la primera línea del documento después de <!DOCTYPE> como su primer elemento, sin embargo, puede estar precedido por comentarios.

Definition

The <html> element is the root element of an HTML document. Usually it is the very first line of the document, after <!DOCTYPE> as it’s the first element, however it can be preceded by comments.

¿Cómo recordar?

Recuerde siempre poner esto primero para identificar lo que está haciendo. Está haciendo un documento HTML así que ponga <html> primero.

How to Remember

Always remember to put this first by identifying what you are making. You’re making an HTML document so put <html> first.

    
      <html>
        <head>
          <title>Document title</title>
        </head>
      </html>
